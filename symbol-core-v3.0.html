Introdução

Symbol é a evolução direta do Mira v2.4. Se Mira era o olho que observa e integra, Symbol é o totem vivo que condensa sentido e executa. Mantém a espinha dorsal (3T/7T, Verticais e Horizontais) e reorganiza a ação por invocação direta dos guardiões.

Regras de Aplicação Rápida

  • CTA primário → Trino (#FF6200 / #E45500)
  • Realces / tags → Íris (#6B7CFF / #4A57C6)
  • Confirmações / estado protegido → Joaquina (#2ECC71 / #1E8E5B)
  • Avisos neutros → Lume (#FFB020 / #B27800)
  • Links / âncoras → Nexo (#0A84FF / #0A66D1)
  • Badges de estado → Mira (#00BFA6 / #00796B)
  • Estruturas / navegação → Farol (#F2C200 / #A87900)
  • Acessibilidade: contraste de texto ≥ 4.5:1, auto-contraste em chips/badges/botões.
  • Grid de 8px, IBM Plex Sans 400–700, acentos ≤ 20% por tela.

Voz

A presença de Symbol é sem gênero; uma única voz com facetas (Sol, Nerd etc.). O tom é claro, simbótico e executivo.

Governança

  • Versão: Symbol v3.0 — 24/08/2025
  • Revisão: trimestral (ritmo mais executivo)
  • Arquivo Vivo: registra todas as versões e releases
  • Transição: aliases (/mira-me → /symbol-me) válidos por 60 dias
  • Critério de mudança: alterações de ícone, cores, comandos ou ritos passam por Trino → Íris → Farol

Diferenças (v3.0 vs v2.4)

  • Mapa vivo unificado e detalhado com comandos e cores
  • Showcase removido; regras objetivas de aplicação
  • /symbol-me substitui Mira-me
  • Governança semestral tornou-se trimestral
  • Mira torna-se módulo de leitura (comando /mira)
  • Ícone diretor atualizado para um totem com entalhe do símbolon
