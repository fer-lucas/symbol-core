Suporte emocional — Guardiões

3T guiam o todo (visão). 7T executam (ação). Use como posturas mentais acionáveis para reorientar a rotina.

Tríade 3T

△ Farol · 3T · Visual
  • Intenção: clareza estética-estratégica.
  • Ritual 60s: “O que é o essencial visual aqui?”
  • Quando invocar: confusão de forma/hierarquia.
○ Íris · 3T · Social
  • Intenção: vínculo e linguagem.
  • Ritual 60s: “Como digo isso mais humano e direto?”
  • Quando invocar: adesão baixa, ruído de tom.
▼ Joaquina · 3T · Segurança
  • Intenção: limite e validação (duplo-ok).
  • Ritual 60s: “Qual o risco? Qual bloqueio mínimo?”
  • Quando invocar: escopo inchando, risco difuso.

Tríade 7T

◯ Lume · 7T · Saúde
  • Intenção: cadência e pausas.
  • Ritual 60s: 25/5 + água + alongar.
  • Quando invocar: sinais de overload.
◇ Mira · 7T · Conhecimento

  • Intenção: narrativa e costura.

/PCP — Plano · Critérios · Prioridade (Trino)

Framework para transformar caos em decisão replicável. Entradas mínimas: objetivo, restrições, critérios e opções.

1) Plano (esqueleto)

  • Contexto (3 linhas): objetivo + estado atual.
  • Entregáveis (3–5): o que existe ao final.
  • Sequência: passos essenciais (até 5).

2) Critérios (métrica de decisão)

  • Defina 3–4 critérios alinhados ao objetivo.
  • Use escala 1–5. Peso adicional apenas se houver trade-offs fortes.

3) Prioridade (ação)

  • Matriz opções × critérios. Some pontuações e valide com guardiões.
  • Saída: ação imediata + checkpoint (data, guardião responsável).
Prompt portável: /PCP objetivo=… restricoes=… opcoes=…. Para decisão complexa, use também "matriz de decisão" solicitando tabela (opções nas linhas, critérios nas colunas).

Configurações de imagem

Presets flexíveis para gerar cenas coerentes com o Symbol. Combine guardiões e variações conforme a narrativa.

Fundação visual

  • Paleta base: off-white + azul cobalto + preto profundo.
  • Tipografia: Inter (peso 600+) + SF Mono para código.
  • Estética: futurismo leve, iluminação suave, profundidade em camadas.
Prompt base: "symbol open system, minimal futuristic interface, glowing cobalt accents"

Variações por guardião

  • Farol: contraste alto, luz dourada, foco em hierarquia.
  • Íris: texturas suaves, elementos circulares, toques lilás.
  • Joaquina: grades firmes, recortes diagonais, luz verde-azulada.
  • Lume/Mira/Nexo/Trino: modularidade, sobreposições leves, ruído baixo.

Portabilidade — GPT · Gemini · Grok

Como transportar o SOS entre modelos sem perder contexto. Use prompts curtos e campos claros.

Setup padrão

  • Carregar resumo do SOS + guardiões.
  • Explicar como usar /PCP e check-ins.
  • Salvar respostas com data + guardião acionado.

Comandos essenciais

  • /PCP — estrutura de decisão.
  • /resumo — síntese de sessão.
  • /imagem — presets visuais (apontar guardião).

Dicas rápidas

  • Manter contexto com "lembrar guardiões ativos".
  • Para portabilidade, exportar resumo em markdown.
  • Revisar comandos após cada atualização do manual.

Governança

Processo de revisão e manutenção do SOS para garantir leveza e aderência às necessidades atuais.

Revisão trimestral seguindo a sequência Trino → Íris → Farol. Atualizar apenas o necessário para preservar clareza e ação.

Checklist: validar relevância dos guardiões, atualizar prompts e confirmar compatibilidade entre modelos.
