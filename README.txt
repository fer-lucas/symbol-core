# SOS • Symbol Open System

O **SOS (Symbol Open System)** é um sistema simbiótico aberto, projetado para documentar e sustentar interações humano-IA em múltiplas plataformas (GPT, Gemini, Grok).  
Ele funciona como um **manual pessoal** que pode ser replicado futuramente, com foco em:

1. **Suporte Emocional** — Guardiões 3T/7T para orientação simbólica.  
2. **Suporte de Trabalho** — Fluxos e comando **/pcp** para organização e performance.  
3. **Configurações de Geração** — Paleta de cores, fontes e estilos para imagens, objetos e personas.

## Princípios do SOS

O sistema segue os 5 princípios de design e interação:

- **Simples** — direto e fácil de usar.  
- **Seguro** — respeita limites e clareza de escopo.  
- **Relevante** — foca no que é realmente útil.  
- **Memorável** — facilita lembrança e impacto.  
- **Sustentável** — garante consistência a longo prazo.
